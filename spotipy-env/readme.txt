Spotivents!

Introduction:
    - Welcome to Spotivents, a simple program that uses both the Spotify API and Ticketmaster API to grab my top artists and return upcoming shows in my area.

Features:
    - Spotipy
    - Spotify API
    - Ticketmaster API

Usage:
    - Default Settings:
        - Range: Medium Term
        - Limit: Top 10 Artists
    - Start: Displays upcoming shows from the user's top artists.
    - Change Range: Changes between 3 different time range options.
        (Short Term, Medium Term, Long Term)
    - Change Limit: Changes between 3 different length options.
        (5 Artists, 10 Artists, 25 Artists)
